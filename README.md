
# 🖼️ Image Captioning AI

## 📌 Project Overview
This project combines Computer Vision and NLP to generate captions for images.

### 🔹 Architecture
Image → ResNet50 (Feature Extraction) → LSTM → Caption Generation

## 🚀 Features
- Pretrained ResNet50 for image feature extraction
- LSTM-based caption generator
- Modular code structure
- GitHub-ready format

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Use feature_extractor.py to extract image features
3. Train model using caption_model.py
4. Generate captions using predict.py

## 📚 Dataset Suggestion
Use Flickr8k or Flickr30k dataset for training.
